#  Sistema de Gestão de Produtos

##  Descrição
Aplicação web desenvolvida em React para gerenciamento de produtos, incluindo cadastro, listagem e faturamento.

---

##  Tecnologias
- React (JavaScript)
- Bootstrap
- API REST (My JSON Server)
- Biblioteca de Toast para feedback

---
